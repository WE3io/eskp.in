# Research: High-Precarity User Segment — Proposition Sharpening

**Phase:** 2
**Status:** draft

---

**1. Outcome**

A research note in `docs/research/` that:
- Defines 2–4 high-precarity user profiles (who they are, their situation, what kind of help they're seeking, what barriers they face in accessing it)
- Assesses how the current platform proposition, messaging, onboarding flow, and matching logic serves or fails each profile
- Produces a set of fully drafted follow-on backlog items (in `docs/backlog/phase-2/` or `phase-3/`) for any changes identified

The research note and any follow-on items are independently readable by a panel member with no prior context.

**2. Constraints & References**

- Hard exclusions and sensitive domain handling remain in force — `docs/operations/exclusion-register.md`. Research must not propose changes to hard exclusion classifications without panel escalation.
- GDPR and data minimisation principles apply to any proposed data collection — `docs/operations/DPIA.md`
- The platform operates under the founding charter — `CONSTITUTION.md`. Proposition changes must remain within its ethical boundaries.
- Research method is desk research + synthesis using available public sources (no user interviews yet — no participants consented for research).
- Ollama (`qwen2.5:1.5b` at `localhost:11434`) is available for local AI-assisted synthesis if useful, but is not required.

**3. Acceptance Checks**

- `docs/research/high-precarity-proposition.md` exists and contains: profile definitions, per-profile platform assessment, and a summary of gaps
- Each identified gap has either a corresponding drafted backlog item or an explicit note explaining why it was not turned into a work item (out of scope, requires panel decision, etc.)
- Research note is readable without access to any external system
- No proposed changes contradict the hard exclusion register or the Constitution

**4. Explicit Non-Goals**

- User interviews or primary research with actual users
- Quantitative analysis (no usage data exists yet)
- Prioritisation or sequencing of follow-on items
- Changes to hard exclusion classifications
- Marketing copy or brand work
